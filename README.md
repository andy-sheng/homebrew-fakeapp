# homebrew-fakeapp

Homebrew tap for [fakeapp](https://github.com/andy-sheng/fakeapp) — build a
debuggable Xcode project from a decrypted iOS IPA.

```sh
brew install andy-sheng/fakeapp/fakeapp
# or
brew tap andy-sheng/fakeapp
brew install fakeapp
```

Try the tip of the default branch without a tagged release:

```sh
brew install --HEAD andy-sheng/fakeapp/fakeapp
```
